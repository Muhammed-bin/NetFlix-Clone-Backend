# Video Streaming Backend

This is a backend project for a video streaming application built using **Node.js**, **Express**, and **MySQL**.

---

## Requirements
- Node.js
- MySQL

---

## How to Run the Project

1. Clone the project
2. Rename `sample.env` to `.env`
3. Add all required credentials in the `.env` file

Run the following commands:

```bash
npm install
npm run dev
